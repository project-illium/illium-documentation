Offline Wallet

A common method of securing cryptocurrency is to use offline storage. In this method one keeps their private keys on device that is not connected to the internet, uses that device to sign a transaction, and transfers the signed transaction to an online device for broadcast to the network.

Ilxd has built-in functionality to make offline signing easy.

If you want your offline wallet to be used with a validator there are some extra steps you need to take. In the setup phase you will also need to copy the node's network private key from the offline computer to the online computer.

The network private key not have access to any of your funds. It is used by your node when authenticating connections between other validators and therefor is safe to keep on an online computer. Worst case scenario if your network key is compromised you will not be able to earn validator rewards until you create a new network key and migrate your node over to it.

Offline Computer

To get your node's network key:

$ ilxcli getnewtorkkey

$ ./ilxd --networkkey=08011240dcd8b19d2cc66f0ec613d4b08b7d73682e2e11122c09959a2cc000b99a525acbb562e48ca118db0f24a53cfbae9f6a3a67f863e6031595d643b7d891621ac280
